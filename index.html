💡 How to use the output in Draw.io:
  1. Copy rows directly from an Excel Sheet or a CSV file and paste them below.
  2. Set your row height, click Generate Draw.io XML, and copy the code.
  3. Open your diagram workspace in Draw.io.
  4. In the top menu bar, go to Extras > Edit Diagram.
  5. Completely overwrite the code block inside that text box, then click OK or Apply.
